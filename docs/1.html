1.用qemu跑linux0.11和xv6

用qemu跑linux0.11和xv6

参考资料

  1. xv6 环境搭建
  2. GDB+Qemu调试Linux-0.11的代码
  3. 生成的镜像均可以在Bochs-win32-2.6.11上运行

准备工作

  1. Ubuntu 18.04.5 LTS，软件源是cn-northwest-1.ec2.archive.ubuntu.com
  2. 安装软件：
    • apt install g++，包含gcc7.5
    • apt install cmake
    • apt install gdb
    • apt install qemu
  3. 做MIT.6828的实验6需要打补丁的qemu：安装的时候需要安装这和这解决报错问题

跑xv6

  1. git clone https://github.com/professordeng/xv6-expansion.git
  2. make
  3. make fs.img
  4. make qemu-nox

跑linux0.11

  1. git clone https://github.com/yuanxinyu/Linux-0.11.git
  2. make
  3. 下载hdc-0.11.img
  4. qemu-system-i386 -m 512 -boot a -fda Image -hda hdc-0.11.img -nographic -curses
