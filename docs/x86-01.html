系统调用细节

系统调用细节

知识

  • x86处理器中的特权级检查及Linux系统调用的实现
  • IDT表的item：段选择符 + 处理函数位置偏移 + DPL
  • GDT表的item：段的基址

步骤

  1. int 80；用户态发起系统调用，此时CPL=3
  2. 找到对应IDT表的item，其中DPL=3，所以权限检查通过，获得段选择符__KERNEL_CS，处理函数位置偏移system_call
  3. 因为__KERNEL_CS中的RPL=0，所以特权级发生变化，此时CPL=0（此步骤都是硬件来完成）
  4. 跳转到__KERNEL_CS: system_call执行（先通过__KERNEL_CS在GDT找到段的基址）
